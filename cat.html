Creating an object in JavaScript

objects in javascript are compared to real life objects ,they have behaviours and
properties.these behaviours are considered as methods .in the example of a car its methods would be
start, stop,hoot,while it's properties are color make and model.
creating an object in javascript is done using the following syntax:
//a key eg name and value eg "Diana" enclosed in curly braces. and stored in a variable 'let'
let obj1={name:"Diana",age:21} console.log(obj1); //this would output an array [name:"Diana",age:21] console.log(obj1.name) //"Diana"
